# Observatoire Open Data Territorial

Pipeline data de bout en bout exploitant des APIs publiques françaises
pour produire un dashboard décisionnel sur des indicateurs territoriaux.

## Stack technique
- Python 3.12
- SQL (SQLite)
- REST APIs (data.gouv.fr, INSEE, Geo API)
- Power BI
- Git / GitHub

## Structure du projet
data/        → données brutes et transformées  
src/         → scripts Python  
notebooks/   → exploration et analyse  
sql/         → schémas et requêtes  
docs/        → documentation et captures
